## Typora 软件简介

### 核心特性

Typora 是一款跨平台 Markdown[^1] 编辑器，支持 Windows、Mac 等操作系统，其核心优势包括：

1. 实时预览功能：消除编辑与预览视图切换，直观呈现文档最终效果，聚焦内容创作；
2. 全面格式支持：兼容 GitHub 风格 Markdown，涵盖代码块、表格、列表、脚注、数学公式等扩展功能；
3. 多场景适配：提供自动保存、版本控制、拼写检查等原生功能，支持 PDF、HTML、Word 等多种格式导出；
4. 个性化定制：通过 CSS 文件自定义主题样式，搭配快捷键操作，满足不同用户使用习惯。

### 适用人群

无论是需要编写技术文档的编程开发者、追求高效写作的作家，还是记录学习笔记的学生，Typora 均能通过简洁的操作界面与高效的功能设计，适配多样化的文档编辑需求。

## Typora 1.9.5 安装教程

### 安装准备

#### 软件与工具下载

- Typora 安装包：通过官网（[https://typora.io](https://typora.io/)）或中文官网（[https://typoraio.cn](https://typoraio.cn/)）下载 1.9.5[^2] 版本安装程序；
- 激活工具：下载压缩包 Typroa_Activation_Script.7z（[蓝奏云](https://www.lanzouh.com/i7j9H26waded)，密码：fkmj；或[百度网盘](https://pan.baidu.com/s/1gga2OZ732Xvig1QKn52NHQ)，密码：dcqe），解压后包含 license-gen.exe、node_inject.exe 两个应用程序及 README.txt 说明文件。

#### 环境要求

本教程适用于 Windows 操作系统，有编程基础或 Linux 系统用户可参考 GitHub 开源项目 [NodeInject_Hook_example](https://github.com/DiamondHunters/NodeInject_Hook_example) 自行编译[^3]使用。

### 安装步骤

#### 软件安装与工具解压

1. 运行 Typora 安装包（如 typora-setup-x64.exe），按提示点击 “下一步” 完成默认安装（默认路径：C:\Program Files\Typora），自定义安装路径需记录备用；
2. 解压激活工具压缩包，得到 license-gen.exe 和 node_inject.exe 两个可执行文件。

#### 文件复制与路径配置

将解压后的 **license-gen.exe** 和 **node_inject.exe** 复制到 Typora 安装目录下（默认路径：C:\Program Files\Typora），若安装路径已修改，需粘贴至对应自定义目录。

#### 命令行操作

1. 按下 Windows 键或在搜索框输入 “CMD”，找到命令提示符，右键选择 “以管理员身份运行”；
2. 若 Typora 安装在非系统盘（如 D 盘），输入盘符（如 D:）并回车，切换至对应磁盘；
3. 输入命令 `cd C:\Program Files\Typora`（自定义路径需替换为实际安装目录），回车跳转至 Typora 安装路径；
4. 输入 `node_inject.exe` 并回车，等待系统提示 “done!”，表示工具运行成功；
5. 输入 `license-gen.exe` 并回车，系统将自动生成序列号，记录该序列号备用。

#### 软件激活

1. 启动 Typora 软件，在激活窗口中随意输入有效格式邮箱地址（如 123@qq.com）；
2. 将生成的序列号粘贴至对应输入框，勾选 “我同意终端用户许可协议和隐私协议”；
3. 点击 “激活” 按钮，完成软件激活。

#### 错误处理

1. 若激活过程中因网络原因出现 “链接服务器失败” 提示，点击 “确认” 按钮即可，不影响激活结果；
2. 多次激活失败，可在不运行Typora的情况下运行CMD，获取新的序列号；
3. 下载错版本，删除干净重新下载，重新生成序列号。

### 注意事项

1. 安装完成后请勿直接更新 Typora，否则可能导致激活失效；
2. 若需更新版本，需先彻底卸载旧版本，再按本教程重新安装并激活新版本。
